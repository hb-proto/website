Welcome to hb.sbs!

Hey, thanks for visiting. If you're here from twitter, you're probably wondering what this is about. It's simple, really:

What the HECK should I do with this website?

I bought this domain quite a while ago while hunting for a short domain name.

During my search, I saw that a new gTLD was registerable (.sbs) and there were a few short domain names avaliable. I bought my brother kyle.sbs (it was only like $2), and I bought myself hb.sbs (it was like $15). How often do you get to buy a 2-letter domain with a short TLD!?

A two-letter domain name! I was pretty excited at first. A few days later though, I realized that I had no idea what I'd even use this domain for. My personal website exists under slightknack.dev, and I have a couple of small independent websites for projects I run (e.g. passerine.io), so I'm not exactly in need of a website...

So without a project in need of a website, I have a website in need of a project!

I mean I have some ideas...

Well, I have some ideas. I could...
  • Use it as a custom link shortener, running something like the null-pointer
  • Make a personal OS / search engine / knowledge base thing.
  • Start a minimal clone of glitch.me.
    • To analogize: hb.sbs would be to glitch.me as riju.codes is to repl.it.
    • Anyone could publish simple remixible webapps, editable in browser.
    • Maybe backed by a minimal persistent real-time database or something.
  • I think I'm getting ahead of myself...
